Contains files relevant to the Master's Thesis of Alex Korocencev titled "Design of an Acoustic Levitation Furnace for High Pressure –Temperature Simulations of (Exo)planetary Bodies"

Sorted in 4 folders:
> 3D CAD Renders, 
> Arduino Codes, 
> COMSOL Multiphysics simulation files (deleted computed datasets, so that the file size does not exceed 100 MB), 
> Python Scripts
